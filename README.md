## Installation

Clone the github repo

```
$ git clone https://github.com/developer-pritam/opencv-tools.get
```

Change Directory

```
$ cd opencv-tools
```

### Requirements :

For requirements run following commands:

```
$ python3 -m pip install -r requirements.txt
```

OR

```
$ python -m pip install -r requirements.txt
```
